# Claude Code Task: ZIM Pipeline, Repo Setup & Size Estimates

## Context

You are building the technical infrastructure for the MComzOS offline library. Read `docs/RATIONALE.md` first.

**MComzOS** is an emergency communications OS for Raspberry Pi (ARM64) and x86_64 Debian 12. The library ships as three custom ZIM files plus WikiMed Mini (downloaded from Kiwix).

## Task 1: Size estimates

Estimate the HTML content size of each core ZIM to verify everything fits on a 16 GB SD card alongside the ~4 GB OS.

**Approach:**
1. Search for the file sizes of each source text (Project Gutenberg file sizes, military manual PDFs, etc.)
2. Estimate HTML-rendered size (typically ~1.2x raw text for simple HTML, more for illustrated manuals)
3. Estimate ZIM compression ratio (ZIM uses zstd; typical ratio is ~2:1 for HTML content)
4. Check WikiMed Mini actual download size from https://download.kiwix.org/zim/

**Produce a table:**
```
| ZIM | Item count | Estimated uncompressed | Estimated ZIM size |
|---|---|---|---|
| MComz-Spiritual.zim | X items | ~X MB | ~X MB |
| MComz-Literature.zim | X items | ~X MB | ~X MB |
| MComz-Survival.zim | X items | ~X MB | ~X MB |
| WikiMed Mini (downloaded) | 1 | ~X MB | ~X MB |
| TOTAL | | | ~X MB |
```

Save as `docs/SIZE_ESTIMATES.md`

## Task 2: Research ZIM compilation tools

Research the current best approach for building custom ZIM files from HTML content:

1. What tools are available? (`zimwriterfs`, `zim-tools`, `zimcreator`, Zimit, etc.)
2. Which is best for our use case (building from local HTML/EPUB files, not scraping websites)?
3. What are the dependencies? Do they work on ARM64?
4. What's the directory structure expected by the ZIM tools?
5. How does Kiwix index/search within a ZIM?

Document findings in `docs/ZIM_BUILD_GUIDE.md`

## Task 3: Create build scripts

Create the scaffolding for building ZIMs:

### `scripts/download-sources.sh`
Script to download all public domain source texts from Project Gutenberg, Internet Archive, etc.

```bash
#!/bin/bash
# Downloads all PD source texts for MComzLibrary
# Usage: ./scripts/download-sources.sh

SOURCES_DIR="sources"
mkdir -p "$SOURCES_DIR/spiritual" "$SOURCES_DIR/literature" "$SOURCES_DIR/survival"

# Example:
# wget -O "$SOURCES_DIR/spiritual/berean-standard-bible.epub" "https://..."
# wget -O "$SOURCES_DIR/literature/sherlock-holmes.epub" "https://standardebooks.org/..."
```

Fill in actual download URLs for every 🟢 CORE item in the RATIONALE.

### `scripts/convert-to-html.sh`
Script to convert EPUBs and PDFs to HTML suitable for ZIM packaging.

### `scripts/build-zims.sh`
Script to compile the three ZIM files from the HTML content.

### `scripts/verify-licences.sh`
Script that outputs a checklist of all included items and their licence status, for human review before distribution.
